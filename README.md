# Job Portal Admin Dashboard

The administrative control panel for the Job Portal system, designed to manage job listings and applicant data.

## Features
- Job Management: Full CRUD operations for creating and maintaining job listings.
- Applicant Management: Centralized interface to view and track all job applicants.
- System Migrations: This repository serves as the primary source for the system's database schema and migrations.

## Technical Stack
- Framework: Laravel 11
- Authentication: Laravel Breeze
- Database:  MariaDB
